## Getting started

아래 명령어를 수행해주세요!

```
$ git clone https://github.com/nogal2/flutter-sample.git
$ cd flutter-sample
$ flutter pub get
$ dart pub run melos bs
$ cd packages/flutter_sample
$ flutter run -d chrome
```
